Welcome to the Digital Industrial Revolution

The Digital Industrial Revolution


Gone are the days when the Factory powered the Middle Class. The world has changed and the road to economic opportunity has changed with it. Yet while technological change has closed one door it has also opened another. The process of building software or coding, has changed dramatically. Coding has gone from a theoretical concept debated by mathematicians to a process by which anyone can learn to build powerful software applications. At Indigenous Code we understand that the ability to create software applications is a tremendous gift. One that has only been available for a short period of time, and that has until now been accessible to only a select few. We aim to correct this problem and spread the power of coding everywhere in the world.


Indigenous Code


A network of Agile Development teams started by a team of Native Americans and Disabled Veterans, Indigenous Code is a both a Community and a Movement. Its mission is to use coding to empower those who have been disempowered.


By spreading the knowledge of coding and developing a large network of Agile Teams, Indigenous Code hopes to ensure a better world tomorrow.
